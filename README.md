# macOS 桌面虚拟宠物 - 产品需求文档

## 项目概述
* 浮灵 - AI 驱动的异世界灵宠
* 简介：一个融合 AI 大模型能力的跨平台桌面虚拟宠物应用，基于 Electron 构建，支持 macOS、Windows、Linux。宠物不仅能在屏幕上自由活动、提供陪伴，更具备智能对话能力，可以倾听你的心情、给予建议、进行情感交流。采用情绪共鸣体（ERE）世界观设定，通过情绪能量系统实现宠物状态管理，支持喂食、玩耍、休眠等互动，具备成长养成、番茄工作法提醒等实用功能，让 AI 陪伴更有温度。

## 核心功能

一个轻量级的跨平台桌面虚拟宠物应用,采用 Electron + Node.js 技术栈开发，支持 macOS、Windows、Linux。核心创新在于将 AI 大语言模型深度集成到宠物交互中，让宠物不只是可爱的动画形象，更是能够理解情绪、进行智能对话的 AI 伙伴。
核心特性
1. AI 智能对话与陪伴

    * 深度 AI 集成：接入大语言模型，宠物具备真实对话能力
    * 状态识别理解：识别用户电脑状态，转换不同形态，安静形态
    * 个性化对话：基于宠物状态（心情、能量）调整对话风格和语气
    * 多场景支持：工作倾诉、生活闲聊、学习陪伴、情绪疏导
    * 治愈系互动：可爱形象 + 智能对话，双重治愈体验

2. 情绪能量系统

    * 基于浮岛世界观的情绪共鸣体（ERE）设定
    * 动态情绪状态：hunger（饥饿度）、energy（能量）、mood（心情）
    * 支持喂食、玩耍、休眠等互动

3. 智能桌面交互

    * 宠物在屏幕上自由移动，支持拖拽
    * 菜单栏集成，快捷操作
    * 多种动画状态（站立、走路、睡觉、互动等）

4. 工作辅助功能

    * 番茄工作法计时提醒
    * 长时间工作后的休息提醒
        * 提醒运动

    * 待办事项提醒
        * 快捷键截图生成待办，定时提醒


5. 局域网互动功能

    * 局域网内发现伙伴
    * 投喂他人灵宠
    * 与他人的灵宠聊天
